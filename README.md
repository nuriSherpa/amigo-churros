# 🍫 Amigo Churros Restaurant Website

A delicious, mobile-friendly website for Amigo Churros, a restaurant specializing in authentic Spanish-style churros and dipping sauces.

## 🛠️ Built With
- PHP - Backend logic and templating
- HTML5 - Structure
- CSS3 - Styling and animations
- JavaScript - Interactive elements

## 🚀 Features
- Responsive design for all devices
- Interactive menu display
- Online ordering system
- Location and contact information
- Photo gallery of our churros

## 🌐 Live Site
Visit the live website: [amigochurros.com](https://amigochurros.com)
